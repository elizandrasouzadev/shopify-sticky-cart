# Shopify Sticky Add to Cart

Página de produto com sticky add to cart — padrão de UX comum em lojas Shopify de alta conversão.

## O que foi feito
- Barra fixa que aparece automaticamente quando o botão de compra sai da viewport
- Galeria de imagens com troca por fade suave ao clicar nas miniaturas
- Feedback visual no botão ao adicionar ao carrinho
- Layout responsivo de página de produto

## Tecnologias
HTML · CSS · JavaScript
